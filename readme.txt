Daily Incident and Workorder Report Generator
_____________________________________________
Experimental, by Souvik Das S0D7	Please reach out in case of any suggestion/complaints/observations
_____________________________________________
1. Keep the .exe file in same folder as :
	a> Incidents.xlsx - the Incident report for the day renamed
	b> Work Orders.xlsx - the Work Order report for the day renamed
	c> Sample.xlsx - Click on the button called 'Template' to download this
2. Run the .exe file
3. report.xlsx will be generated - that is the result to be renamed and used
_____________________________________________
Note :
1. Give the application a few seconds to run - it might not show but it is working in the background
2. Do not keep the Incidents.xlsx or Work Orders.xlsx or report.xlsx open while the .exe file works
3. report.xlsx may be deleted, does not matter - a fresh file is created every time the executable file is run
_____________________________________________
Need no dependency
